Laboratoires

  1. Installation de Xcode sur macOS
    1. Accéder à l’App Store
      • Ouvrez l’App Store sur votre Mac (en utilisant le Dock ou en recherchant “App Store” via Spotlight).
    2. Rechercher Xcode
      • Tapez “Xcode” dans la barre de recherche de l’App Store.
    3. Télécharger Xcode
      • Cliquez sur “Obtenir” ou “Installer” pour télécharger Xcode. Selon votre connexion Internet, l’installation peut prendre un certain temps (Xcode est un fichier volumineux).
    4. Lancer Xcode
      • Une fois l’installation terminée, vous pouvez ouvrir Xcode depuis le dossier Applications ou en le recherchant via Spotlight (Cmd + Espace, puis tapez “Xcode”).
  2. Se familiariser avec l’environnement macOS
    1. Découvrir le Finder
      • Le Finder est l’outil principal pour naviguer dans les fichiers de macOS. Il est accessible depuis l’icône en forme de visage souriant dans le Dock.
      • Vous pouvez l’utiliser pour ouvrir des fichiers, des dossiers et accéder à vos applications.
    2. Utiliser Spotlight pour rechercher rapidement
      • Appuyez sur Cmd + Espace pour ouvrir Spotlight, qui vous permet de rechercher des fichiers, des applications ou d’autres contenus sur votre Mac.
    3. Explorer le Dock
      • Le Dock en bas de votre écran contient des icônes d’applications fréquemment utilisées. Vous pouvez y ajouter ou supprimer des applications en faisant glisser les icônes.
    4. Se familiariser avec les Préférences Système
      • Allez dans Préférences Système pour personnaliser votre Mac, configurer votre réseau, votre affichage, et d’autres paramètres de l’ordinateur.
